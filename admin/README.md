# Home Soluciones - Panel de Administración

Este es el CMS personalizado para Home Soluciones, construido con React, Vite y Framer Motion.

## Características

- **Diseño Premium**: Interfaz moderna con estética "Obsidiano", modo oscuro y efectos de glassmorphism.
- **Gestión de Landing**: Control total sobre el Hero, estadísticas, proyectos y procesos.
- **Gestión de Servicios**: Módulo para administrar el catálogo de servicios.
- **Blog**: Panel para publicar y editar entradas del blog.
- **Responsive**: Totalmente adaptado para dispositivos móviles y tablets.

## Tecnologías

- **React 18**
- **Vite**
- **Lucide React** (Iconos)
- **Framer Motion** (Animaciones)
- **Vanilla CSS** (Estilos)

## Desarrollo

Para iniciar el panel en modo desarrollo:

```bash
npm install
npm run dev
```

## Integración con Backend

El CMS está preparado para conectarse con la API de NestJS. Los servicios de datos se encuentran en `src/services` (próximamente).
